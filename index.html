Depression Detection in Singlish Text

Ability to detect depression in Sinhala-English code mixed text posted by people

Get Started

Learn more....

Since there is a difficulty in early depression detection among Sri Lankan people, the depression detection system will be a solution to it by detecting depression based on texts which are in Sinhala-English code-mixed language. This is a research project based on natural language processing and machine learning.

End users of this proposed system would be mental health professionals who will detect depression and help people. The proposed system was tested with live data collected from social platforms and it achieved an accuracy of 74.11% in detecting depression.

  • Depression Detection

    Detects depression in Singlish text and its the core feature of this system

  • Estimated Depression Rate

    Displays the depression percentage for a Singlish text if the text contain depressive content

  • Store Detection Results

    Stores the detection results with the input text for future processing

  • Acceptance of text in different lengths

    Able to accept and process Singlish text in any size of length

Frequently Asked Questions

  • Does this system cure depression as well?

    No, this system only focuses on detecting depression to facilitate early depression detection.

  • Does this support other code-mixed languages?

    For now it doesnt but it is considered as a future enhancement.

  • Can this system process multiple texts at the same time?

    No, it can only process one input at a time.

  • Does this system uses AI to detect depression?

    Yes, it uses a part of AI which is known as machine learning to detect depression.

  • Can this system send alerts to mental healthcare professionals?

    No it doesn't send alerts because the depression detection system has to be integrated with mental healthcare services.
